  • J.A.R.V.I.S


    Jarvis is a Python-based virtual assistant designed to automate daily tasks with innovative features. It includes secure face recognition for user authentication, voice command execution for opening apps, controlling media, and retrieving information. JARVIS manages tasks, sets alarms, and plays music on Spotify. It uses OCR to scan screen content and AI APIs to generate solutions for programming queries. With capabilities like scheduling, system control, and AI-driven problem-solving, JARVIS demonstrates expertise in AI, automation, and natural language processing.



  • HUMAN ACTIVITY RECOGNITION


    The Human Activity Recognition project uses time-series data to identify human activities. The academic version was built using an LSTM Recurrent Neural Network (RNN) with two LSTM layers and two fully connected layers, each with 64 units, processing 200-sample sequences. Data was preprocessed into smaller time slices for better analysis. The updated version improved this by using larger and more diverse datasets, optimizing the model through better tuning and architecture adjustments, which led to higher accuracy and better performance. This progression highlights significant improvements in activity recognition techniques.



  • MOVIE RECOMMENDATION SYSTEM


    Developed a personalized Movie Recommendation System using R, leveraging collaborative filtering techniques to suggest movies based on user preferences and historical data. Key functionalities include data preprocessing, genre-based filtering, and user-item similarity analysis using cosine metrics. Implemented Item-Based Collaborative Filtering (IBCF) with the recommenderlab library to predict top recommendations. Enhanced data visualization through heatmaps, bar plots, and distribution charts to illustrate patterns and trends. The system effectively demonstrates expertise in data analysis, machine learning, and recommendation model development.
